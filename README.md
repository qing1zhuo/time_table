# 周计划模板 (Weekly Plan Template)

这是一个轻量级的本地桌面应用，旨在帮助您管理一个固定的周计划模板。它就像一张电子课程表，一次设定，永久有效，并会在指定时间通过桌面通知提醒您。

## ✨ 功能特性

- **模板化管理**：只关注一个标准的周计划，无需处理复杂的多周日期。
- **自动保存**：在单元格中输入计划后，只需将鼠标焦点移开，内容即会自动保存，无需手动操作。
- **桌面提醒**：在每天的 **上午9:00**、**下午14:00** 和 **晚上19:00**，应用会自动检查您当天的计划，并通过系统桌面通知提醒您。
- **无窗口启动**：通过 VBScript 实现静默启动，无黑框打扰。
- **开机自启**：可以轻松配置，让应用在您每次打开电脑时自动在后台运行。
- **环境自适应**：启动脚本会自动检测并安装所需的依赖库，无论是在虚拟环境还是系统全局环境中都能顺利运行。

## 🚀 如何启动

1.  进入项目文件夹 `time_table`。
2.  找到并 **双击 `run.vbs`** 文件。

脚本会自动在后台完成环境检查、依赖安装，并启动主程序窗口，全程无控制台窗口。

## 📝 如何使用

- **编辑计划**：直接在主界面的表格中点击对应的单元格（例如“周一”和“上午”的交叉格），然后输入您的计划内容。
- **保存计划**：**无需手动保存！** 当您完成输入，并将鼠标点击到其他单元格或窗口的其他位置时，内容就会被自动记录到数据库中。

## ⚙️ 如何设置开机自启动

如果您希望应用在每次开机时自动运行，请按照以下步骤操作：

1.  **创建快捷方式**：
    -   在项目文件夹中，找到 `run.vbs` 文件。
    -   右键单击它，然后选择 **“发送到”** -> **“桌面(创建快捷方式)”**。
2.  **打开“启动”文件夹**：
    -   按下键盘上的 **Win + R** 组合键，打开“运行”对话框。
    -   输入 `shell:startup` 并按“确定”。
3.  **移动快捷方式**：
    -   将您刚刚在桌面上创建的快捷方式，拖动或复制到这个“启动”文件夹中。

完成！现在，您的周计划模板应用将会在每次您登录Windows时自动在后台启动，并准时提醒您。

## 📂 文件结构

```
time-plan/
├── backend/                # 后端模块
│   └── db.py               # 数据库初始化和操作逻辑
├── db/
│   └── plan.db             # SQLite 数据库文件 (自动生成)
├── .venv/                  # Python 虚拟环境 (推荐)
├── main.pyw                # 应用主程序 (GUI界面和提醒逻辑)
├── requirements.txt        # 项目依赖库列表
├── run_desktop_app.bat     # 核心启动脚本 (由run.vbs调用)
└── run.vbs                 # 无窗口静默启动脚本 (推荐使用)
```
